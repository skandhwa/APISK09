Pass the Bearer Token from Env Variable
Pass "https://gorest.co.in" from Global Variable
Pass "public/v2/users" from Env Variables

Validations
----------------------
Validate the Response code 

Validate Header as Content Type
Validate the name and email which u are sending from pre request script 
Validate the ID generated should always be an integer
Validate the ID generated should not be null.
